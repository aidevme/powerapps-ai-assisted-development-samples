# AI-Powered Canvas App Development: Build Power Apps with GitHub Copilot & Claude Code

## Overview

This repository contains sample code and resources for building Microsoft Power Apps Canvas applications using AI-assisted development tools, specifically **GitHub Copilot** and **Claude Code**. The samples demonstrate how to leverage AI capabilities to accelerate Power Apps development, improve code quality, and streamline the development workflow.

## About

Power Apps development can be significantly enhanced using AI coding assistants. This repository provides practical examples and best practices for:

- Using GitHub Copilot to generate Power Apps formulas and expressions
- Leveraging Claude Code for complex logic and application architecture
- Implementing AI-suggested patterns for common Power Apps scenarios
- Optimizing app performance with AI recommendations
- Automating repetitive development tasks

## Technologies

- **Microsoft Power Apps** - Low-code platform for building business applications
- **GitHub Copilot** - AI pair programmer for code suggestions and completions
- **Claude Code** - Advanced AI assistant for code generation and problem-solving
- **Power Fx** - The low-code language for Power Apps

## Prerequisites

Before using these samples, ensure you have:

- Microsoft Power Apps account (trial or licensed)
- Power Apps Studio (web or desktop)
- GitHub account with Copilot access (optional but recommended)
- Claude API access or Anthropic account (optional but recommended)
- Basic understanding of Power Apps and Power Fx

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/aidevme/powerapps-ai-assisted-development-samples.git
   cd powerapps-ai-assisted-development-samples
   ```

2. **Browse the samples**
   - Navigate to the `samples/` directory to explore different examples
   - Each sample includes a description and implementation details

3. **Import into Power Apps**
   - Open Power Apps Studio
   - Import the sample .msapp files or copy the formulas/code snippets
   - Follow the README in each sample folder for specific instructions

## AI Development Tips

### Using GitHub Copilot
- Provide clear comments describing what you want to achieve
- Use descriptive variable and control names
- Let Copilot suggest formulas as you type

### Using Claude Code
- Describe your requirements in detail
- Ask for explanations of complex formulas
- Request optimization suggestions for existing code
- Use Claude to generate test scenarios
